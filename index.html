Objetivo:

Este proyecto se enfocó en el desarrollo de una plataforma de comparación de Productos utilizando Inteligencia Artificial (IA). La meta principal fue facilitar la decisión entre productos a la hora de hacer una compra, además permitiendo obtener el producto recomendado utilizando la API de Amazon.

Herramientas Utilizadas:

El proyecto se implementó utilizando el lenguaje de programación Python, aprovechando la versatilidad de la biblioteca 'requests', el Framework de Flask para construir la API en el backend y bootstrap para el frontend. Además se utilizó ChatGPT para la recomendación de IA en conjunto de una API de RapidAPI.com para los productos de Amazon.

Por último también se integró algunos controles como Anticaptcha para evitar Bots y CSRFToken para que la herramienta sea consumida desde el frontend.

Funcionalidades Implementadas:

Comparar entre N cantidad de Productos

Obtener Productos de Amazon para hacer Marketing de Afiliados

Obtener 5 ventajas del producto recomendado

Obtener una review de YouTube del producto recomendado

Objetivo:

En este proyecto dedicado a la seguridad informática, se evaluó una plataforma de streaming de playas diseñada para la comunidad de surf. Utilizando técnicas de web scraping, exploramos la infraestructura digital de la plataforma para identificar posibles vulnerabilidades.

Herramientas Utilizadas:

Requests: Se utilizó la librería Requests para realizar solicitudes HTTP, permitiéndonos interactuar con la infraestructura web de manera efectiva y recuperar datos esenciales para la evaluación.

cv2 (OpenCV): La librería cv2 de OpenCV se incorporó para el procesamiento de imágenes. Este componente resultó crucial para analizar y manipular visualmente datos extraídos.

io, BytesIO y tempfile: Estas librerías, específicamente io y BytesIO, fueron esenciales para trabajar con datos en memoria y facilitar la manipulación eficiente de bytes. tempfile también se implementó para gestionar archivos temporales de manera segura durante el proceso.

Consideraciones Éticas y de Seguridad:

Es crucial destacar que el web scraping, especialmente en sitios vinculados con servicios pagos, debe llevarse a cabo de manera ética y cumpliendo con los términos de servicio del sitio web objetivo. Debido a las razones mencionadas anteriormente, el repositorio del proyecto no será hecho público.
